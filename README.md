# SmartStudy AI

SmartStudy AI is a web-based study planner built using Flask, HTML, CSS and JavaScript.

It helps students track their study progress, analyze subject distribution using charts, and interact with an AI study assistant.

## Features

- User Login & Signup
- Study Progress Dashboard
- Subject Distribution Charts
- AI Study Assistant Chatbot
- Study Planner Interface

## Technologies Used

- Python (Flask)
- HTML
- CSS
- JavaScript
- Chart.js

## How to Run

1. Clone the repository
2. Install Flask
3. Run:

python app.py

4. Open browser:
http://127.0.0.1:5000
